# اپلیکیشن استعدادیابی ورزشی (تنیس)

## مقدمه
این پروژه با هدف طراحی و پیاده‌سازی یک اپلیکیشن استعدادیابی ورزشی در رشته تنیس انجام شده است. تمرکز پروژه بر استفاده از مدل‌های ماشین لرنینگ و پردازش تصویر و ویدیو برای تحلیل ویژگی‌های فیزیکی و حرکتی افراد و ارائه یک پیشنهاد اولیه درباره مناسب بودن آن‌ها برای رشته تنیس است.

اپلیکیشن شامل دو بخش اصلی است:
1. بخش متنی (داده‌های عددی کاربر)
2. بخش ویدیویی (تحلیل حرکت فورهند با Pose Detection)

## ساختار کلی پروژه
پروژه از نظر منطقی به سه مرحله تقسیم می‌شود:
- جمع‌آوری ورودی‌های متنی و ویدیویی
- استخراج ویژگی‌ها
- پیش‌بینی با استفاده از مدل‌های ماشین لرنینگ

رابط کاربری با استفاده از کتابخانه Gradio پیاده‌سازی شده است تا کاربر بتواند به‌صورت ساده اطلاعات خود را وارد کرده و ویدیو بارگذاری کند.

## بخش متنی (Text-based Analysis)

### ورودی‌ها
در بخش متنی، اطلاعات زیر از کاربر دریافت می‌شود:
- سن (عدد گسسته)
- قد (سانتی‌متر)
- طول دست‌ها (سانتی‌متر)
- زمان تست چابکی
- زمان عکس‌العمل
- جنسیت

### مدل ماشین لرنینگ
در این بخش از مدل Random Forest Classifier استفاده شده است. این مدل به دلیل عملکرد مناسب روی داده‌های عددی، مقاومت در برابر نویز و عدم نیاز به نرمال‌سازی پیچیده انتخاب شده است.

خروجی مدل یک برچسب باینری است:
- 1 : مناسب برای تنیس
- 0 : پیشنهاد سایر رشته‌های ورزشی

## بخش ویدیویی (Video-based Analysis)

### ورودی ویدیو
کاربر یک ویدیوی کوتاه از حرکت فورهند تنیس بارگذاری می‌کند.

### استخراج ویژگی
برای استخراج ویژگی‌های حرکتی از کتابخانه MediaPipe Pose استفاده شده است. این کتابخانه موقعیت مفاصل اصلی بدن مانند شانه، آرنج و مچ را در هر فریم استخراج می‌کند.

ویژگی‌های مورد استفاده شامل:
- زاویه آرنج دست غالب
- تغییرات زاویه شانه
- پایداری بالاتنه
- هماهنگی حرکت دست و تنه

برای هر ویدیو، میانگین و واریانس این ویژگی‌ها محاسبه شده و به‌عنوان بردار ویژگی نهایی در نظر گرفته می‌شود.

### مدل ماشین لرنینگ ویدیویی
برای تحلیل داده‌های ویدیویی از مدل Random Forest Classifier یا Support Vector Machine استفاده شده است که روی ویژگی‌های استخراج‌شده آموزش داده شده‌اند.

## ترکیب نتایج
نتیجه نهایی اپلیکیشن با ترکیب خروجی بخش متنی و بخش ویدیویی به‌دست می‌آید. اگر هر دو مدل نتیجه مثبت ارائه دهند، فرد به‌عنوان دارای استعداد اولیه برای رشته تنیس معرفی می‌شود.

## رابط کاربری
رابط کاربری با استفاده از Gradio طراحی شده و شامل:
- فیلدهای عددی و اسلایدرها برای اطلاعات متنی
- دکمه بارگذاری ویدیو
- نمایش نتیجه نهایی

## جمع‌بندی
این پروژه نمونه‌ای از کاربرد عملی ماشین لرنینگ و بینایی ماشین در حوزه ورزش است و نشان می‌دهد چگونه می‌توان با ترکیب داده‌های عددی و حرکتی، یک سیستم استعدادیابی اولیه طراحی کرد.
